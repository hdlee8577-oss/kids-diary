# Pathfinder Kids - 비즈니스 모델 요약 및 개선 방향

## 📌 현재 프로젝트 vs 목표 비즈니스 모델

### 현재 상태 ✅
- ✅ 기본 사진/일기 저장소
- ✅ 테마 커스터마이징
- ✅ Supabase 연동

### 목표 상태 🎯
- 🎯 **College Ready 포트폴리오 시스템**
- 🎯 **학년별/분야별 체계적 분류**
- 🎯 **성장 스토리텔링 도구**

---

## 🔑 핵심 추가 기능 (우선순위별)

### 1순위: 구조적 분류 시스템 ⭐⭐⭐
**왜 중요한가?** 미국 대입에서 가장 중요한 것은 "체계적인 활동 기록"

**추가할 것:**
- 학년 선택 (Pre-K ~ 12th)
- 분야 분류 (STEM, Art, Sports, Writing)
- 태그 시스템
- 필터링 UI

**비즈니스 가치:**
- 대입 에세이 작성 시 "3학년부터 10학년까지의 STEM 활동" 같은 구체적 증거 제공
- 포트폴리오 자동 구성 가능

---

### 2순위: "Mom's Note" 기능 ⭐⭐⭐
**왜 중요한가?** 아이가 나중에 봤을 때 맥락을 이해할 수 있음

**추가할 것:**
- 각 작품/일기에 엄마의 코멘트 추가
- "이 날 우리 아이가 처음으로..." 같은 스토리 기록

**비즈니스 가치:**
- 감성적 연결고리
- 나중에 "성장 스토리" 콘텐츠로 활용 가능

---

### 3순위: 검색 기능 ⭐⭐
**왜 중요한가?** 수년간 쌓인 데이터에서 빠르게 찾기

**추가할 것:**
- 제목, 태그, Mom's Note 검색
- OCR 텍스트 검색 (나중에)

---

### 4순위: 성장 타임라인 비교 ⭐⭐
**왜 중요한가?** "성장"을 시각적으로 보여줌

**추가할 것:**
- 같은 주제(예: 자화상)의 작품을 시간순으로 비교
- "1학년 vs 5학년" 비교 뷰

**비즈니스 가치:**
- 유튜브 콘텐츠로 활용 가능
- "우리 아이 성장 스토리" 영상 제작

---

### 5순위: PDF 포트폴리오 내보내기 ⭐
**왜 중요한가?** 대학/경시대회 제출용

**추가할 것:**
- 선택한 작품들을 PDF로 생성
- 커스터마이징 가능한 레이아웃

**수익 모델:**
- Freemium: 기본 기능 무료, PDF 내보내기는 유료

---

### 6순위: AI OCR (손글씨 인식) ⭐
**왜 중요한가?** 아이가 쓴 손글씨를 검색 가능하게

**추가할 것:**
- 이미지 업로드 시 OCR 옵션
- 손글씨 → 텍스트 변환
- 검색에 포함

**기술 선택:**
- Google Cloud Vision API (한글 우수)
- 또는 Naver Clova OCR

---

### 7순위: 프라이빗 공유 ⭐
**왜 중요한가?** 조부모님, 친척과 안전하게 공유

**추가할 것:**
- 특정 앨범에 대한 공유 링크 생성
- 만료일 설정
- 비밀번호 옵션

---

## 🏗️ 구조 개선 사항

### 현재 구조
```
/ (홈)
/photos (사진첩)
/diary (일기장)
```

### 개선된 구조
```
/ (대시보드 - 학년별/분야별 요약)
  - "1학년: 15개 작품, 3개 일기"
  - "STEM: 8개, Art: 12개"
  - 최근 업로드

/portfolio (포트폴리오 뷰)
  /portfolio/grade/[grade] (학년별)
  /portfolio/category/[category] (분야별)
  /portfolio/timeline/[tag] (타임라인 비교)

/photos (사진첩)
  /photos?grade=1st&category=Art (필터링)

/diary (일기장)
  /diary?grade=2nd&category=Writing

/upload (통합 업로드)
  - 사진/작품 업로드
  - 일기 작성
  - 학년, 분야, 태그 선택

/share/[token] (공유 링크)
```

## 💰 수익 모델 연결

### Freemium 모델
- **무료:** 기본 저장, 기본 분류
- **유료 ($9.99/mo):**
  - 대용량 저장
  - PDF 내보내기
  - 고급 필터링
  - OCR 기능

### B2B 모델 (나중에)
- 미술 학원, 사립 학교에 학생 관리 솔루션으로 판매
