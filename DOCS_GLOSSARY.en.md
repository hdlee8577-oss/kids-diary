## Documentation Glossary (US English)

This glossary standardizes terms across the English documentation set.

### Product modules
- **Photo Album**: `photos` module (Korean: 사진첩)
- **Diary**: `diary` module (Korean: 일기장)
- **Artworks**: `artworks` module (Korean: 작품/작품 모음)
- **Timeline**: unified view across modules (Korean: 타임라인)
- **Stats**: analytics and trends (Korean: 통계)
- **Settings**: configuration panels (Korean: 설정)

### Platform / architecture
- **Theme**: real-time UI theme system (Korean: 테마)
- **Migration**: database schema update (Korean: 마이그레이션)
- **Admin token**: optional token used to protect write endpoints (Korean: 관리자 토큰)
- **Site / siteId**: logical tenant identifier (default: `siteConfig.siteId`)

### Conventions
- **Date format**: `YYYY-MM-DD`
- **Time zone**: dates are treated as local dates when applicable (avoid UTC shifts)
