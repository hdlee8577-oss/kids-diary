## ✅ 좋은 요청 예시

### 예시 1: 명확하고 구체적
```
"커스터마이즈 시스템의 메뉴 모듈 정의 파일을 만들어줘.
menuModules.ts 파일에 모든 메뉴 모듈을 정의하고,
각 모듈에는 id, label, icon, path, description, category가 포함되어야 해."
```

### 예시 2: 단계별 요청
```
"1단계: DB 스키마부터 만들어줘
- user_menu_settings 테이블 생성 SQL
- RLS 설정 포함
- docs/supabase-menu-settings.sql 파일로 저장"
```

### 예시 3: 참고 파일 명시
```
"IMPLEMENTATION_GUIDE.md 파일을 참고해서
업로드 폼에 학년, 분야, 태그 필드를 추가해줘"
```

## 🎯 단계별 작업 요청 예시

### Week 1: 커스터마이즈 시스템 기반

#### 작업 1: 메뉴 모듈 정의
```
"커스터마이즈 시스템의 메뉴 모듈 정의 파일을 만들어줘.

CUSTOMIZATION_SYSTEM.md를 참고해서:
1. src/config/menuModules.ts 파일 생성
2. 모든 메뉴 모듈 정의 (photos, diary, artworks, awards 등)
3. 각 모듈에 id, label, icon, path, description, category 포함
4. TypeScript 타입 정의 포함"
```

#### 작업 2: 프리셋 정의
```
"메뉴 프리셋 정의 파일을 만들어줘.

1. src/config/menuPresets.ts 파일 생성
2. 아기, 유아, 유치원, 초등, 중고등 프리셋 정의
3. 각 프리셋에 enabledModules, ageRange 포함"
```

#### 작업 3: DB 스키마
```
"메뉴 설정을 위한 DB 스키마를 만들어줘.

1. docs/supabase-menu-settings.sql 파일 생성
2. user_menu_settings 테이블 생성
3. RLS 설정 포함
4. 인덱스 추가"
```

## 💡 효과적인 작업 요청 팁

### 1. 한 번에 하나씩
✅ 좋음: "메뉴 모듈 정의 파일 만들어줘"
❌ 나쁨: "커스터마이즈 시스템 다 만들어줘"

### 2. 구체적으로
✅ 좋음: "업로드 폼에 학년 선택 드롭다운 추가해줘"
❌ 나쁨: "업로드 폼 개선해줘"

### 3. 참고 파일 명시
✅ 좋음: "IMPLEMENTATION_GUIDE.md 참고해서..."
❌ 나쁨: (참고 없이 요청)

### 4. 예상 결과 명시
✅ 좋음: "저장하면 DB에 저장되고 화면에 반영되어야 해"
❌ 나쁨: "저장 기능 만들어줘"

### 5. 우선순위 명시
✅ 좋음: "이건 급한데, 나머지는 나중에 해도 돼"
❌ 나쁨: (우선순위 없이 모든 것을 급하게)

## 🚀 실제 작업 요청 예시

### 예시 1: 메뉴 모듈 정의
```
"커스터마이즈 시스템의 첫 번째 단계로 메뉴 모듈 정의 파일을 만들어줘.

CUSTOMIZATION_SYSTEM.md의 '1. 메뉴 모듈 정의' 섹션을 참고해서:
- src/config/menuModules.ts 파일 생성
- MenuModuleId 타입 정의
- MenuModule 타입 정의
- MENU_MODULES 배열에 모든 메뉴 정의
- photos, diary는 이미 있으니 참고해서 구조 맞춰줘"
```

### 예시 2: DB 스키마
```
"메뉴 설정을 저장할 DB 테이블을 만들어줘.

docs/supabase-menu-settings.sql 파일 생성:
- user_menu_settings 테이블
- enabled_modules (JSONB)
- menu_order (JSONB)
- role_mode, age_in_months, preset 필드
- RLS 설정 포함
- 기존 supabase-site-settings.sql 파일 구조 참고"
```

### 예시 3: 설정 페이지
```
"메뉴 설정 페이지를 만들어줘.

/src/app/settings/menu/page.tsx 파일 생성:
- 프리셋 선택 UI (그리드 레이아웃)
- 메뉴 직접 선택 (체크박스)
- 카테고리별 그룹화
- 저장 버튼
- CUSTOMIZATION_SYSTEM.md의 '5. 메뉴 설정 페이지' 참고"
```
