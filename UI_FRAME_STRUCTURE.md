## 📱 각 페이지별 UI 프레임

### 1. 홈페이지 (`/`)

```
┌─────────────────────────────────────┐
│  [헤더]                             │
├─────────────────────────────────────┤
│                                     │
│  ┌───────────────────────────────┐ │
│  │ [배경 그라데이션]             │ │
│  │                               │ │
│  │  ┌─────────┐  ┌──────────┐  │ │
│  │  │ 텍스트  │  │ 프로필   │  │ │
│  │  │ 영역    │  │ 사진     │  │ │
│  │  │         │  │          │  │ │
│  │  │ - 배지  │  │ [편집]   │  │ │
│  │  │ - 제목  │  │ 생일정보 │  │ │
│  │  │ - 소개  │  └──────────┘  │ │
│  │  └─────────┘                 │ │
│  │                               │ │
│  │  [버튼 영역]                  │ │
│  │  [사진첩 보기] [일기장 쓰기]  │ │
│  │                               │ │
│  │  [기능 소개 카드 3개]         │ │
│  │  ┌──┐ ┌──┐ ┌──┐             │ │
│  │  │📸│ │📝│ │📏│             │ │
│  │  └──┘ └──┘ └──┘             │ │
│  └───────────────────────────────┘ │
│                                     │
└─────────────────────────────────────┘
```

**구성 요소:**
- **배경**: 그라데이션 원형 블러 효과
- **왼쪽**: 텍스트 콘텐츠 (배지, 제목, 소개)
- **오른쪽**: 프로필 사진 (편집 버튼, 생일 정보)
- **하단**: 액션 버튼 + 기능 소개 카드

## 📐 반응형 브레이크포인트

- **모바일**: `< 640px` (sm 미만)
  - 햄버거 메뉴
  - 세로 레이아웃
  - 작은 패딩 (px-4)

- **데스크톱**: `≥ 640px` (sm 이상)
  - 가로 메뉴
  - 큰 패딩 (px-6)
  - 최대 너비 제한 (max-w-5xl)

---

## 🎯 주요 프레임 영역

### 1. **헤더 영역** (고정)
- 높이: `py-3` (약 48px)
- 위치: `sticky top-0`
- 배경: 반투명 + 블러

### 2. **메인 콘텐츠 영역**
- 최대 너비: `max-w-5xl` (1024px)
- 패딩: `px-4` (모바일) / `px-6` (데스크톱)
- 중앙 정렬: `mx-auto`

### 3. **사이드바 영역** (오른쪽)
- 너비: 고정 또는 가변
- 위치: `fixed right-0`
- 토글: 설정 버튼 클릭

### 4. **모바일 메뉴** (오른쪽)
- 너비: `w-64` (256px)
- 위치: `fixed right-0 top-0`
- 전체 높이: `h-full`
